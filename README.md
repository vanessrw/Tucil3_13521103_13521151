## Requirements
- flask
- timeit
- networkx

## Setup

Clone this repo using the command below: 
**git clone https://github.com/vanessrw/Tucil3_13521103_13521151.git**

- `pip install networkx`
- `pip install timeit`
- `pip install flask`
jalankan `py web.py` atau 'python web.py' pada terminal
